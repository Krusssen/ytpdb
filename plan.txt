Функционал:
1. Возможность смотреть каждую базу данных
2. Возможность искать игрока по id
3. Возможность искать поле в любой таблице по произвольному слову
	после чего будет выведен список всех таблиц содержащих это слово
4. Добавление новых полей в любую выбранную таблицу (все поля должны быть
	предусмотрены при вводе без каких либо пропусков)
5. Удаление любой строки (по id)
6. Красивое меню (по желанию блядь)

Проектировние:
1. Для каждой таблицы все пишется через табуляцию
	Пример:
	id	username	class	race	...
	1	PidorGoyda	Paladin	Human	...
2. Каждая таблица записывается в отдельном файле
3. Все файлы с таблицами лежат в /db/...
4. Для каждого функционала создаем отдельный метод. В функции main оставляем
	только то, что будет использовано только 1 раз
